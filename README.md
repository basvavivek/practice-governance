# Municipal Drainage Complaint Portal

This project includes a browser-ready static page that can be opened on another laptop without Node.js.

## Open on another laptop

1. Copy the project folder `municipal drainage` to the other laptop.
2. Open `run.html` in a web browser.

## Notes

- `run.html` uses CDN links for React, ReactDOM, Babel, and Tailwind.
- The other laptop must have internet access to load those CDN resources.
- If you need a fully offline version, I can also create a local static bundle.
